# Cinema Machina Brain Stack

## Purpose
Maintain durable project intelligence for Codex, Antigravity, Claude Code, and future agents.

## Layers

### Graphify
Use for:
- codebase architecture
- module relationships
- repo summaries
- structural navigation
- lower-token repo exploration

Rule:
Use Graphify before broad file scanning.

### Codex Memories
Use for:
- Codex-local recall of stable preferences
- recurring workflows
- project conventions
- repeated pitfalls

### OMEGA
Use for:
- cross-agent persistent memory
- architectural decisions
- stable project constraints
- recurring technical learnings
- reusable debugging lessons

Never store:
- API keys
- passwords
- tokens
- secrets
- private credentials

### Obsidian
Use for:
- curated human notes
- daily logs
- roadmap
- decision records
- risks
- visual summaries

## Working Rule
For repository-level reasoning:
1. Read Graphify outputs.
2. Query Graphify when useful.
3. Retrieve relevant persistent memory.
4. Open only exact source files needed.
5. Record stable conclusions in the brain layer.
