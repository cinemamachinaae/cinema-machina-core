# Toolchain Routing

## Codex
Primary coding assistant for Cinema Machina development.

Use:
- Graphify first for repo understanding
- OMEGA when prior decisions or durable memory may help
- Codex Memories for stable local recall
- exact-file reading only after graph/memory review

## Langflow
Use later for:
- visual AI workflow prototyping
- reusable tool chains
- MCP-ready flows

Langflow does not automatically improve Codex until useful flows are exposed as MCP-accessible tools.

## RuFlo
Use later for:
- multi-agent orchestration
- swarm-style task breakdown
- complex coordinated workflows

Do not use RuFlo as the default daily path for ordinary Cinema Machina coding tasks.

## Obsidian
Use as the human-readable Cinema Machina Brain:
- decisions
- roadmaps
- risks
- daily logs
- high-level summaries
